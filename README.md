# RGB-LED-Strip-Controller-for-Arduino-Nano
Проект представляет собой контроллер на базе Arduino Nano для управления светодиодной RGB лентой с общим анодом и напряжением 12–24В. Устройство позволяет вручную настраивать цвета и автоматически включать их по расписанию на каждый день недели.

###  Аппаратные возможности
*   Управление лентой через MOSFET транзисторы (ШИМ).
*   Высокочастотный программный ШИМ для устранения мерцания (по умолчанию 62.5 кГц, можно менять, см. страницу AlexGyver на github GyverPWM).

### ТАБЛИЦА ЧАСТОТ ШИМ ДЛЯ ARDUINO NANO (16 МГц)

| Режим `mode` | Таймер 1 (пины 9 и 10) 8 bit | Таймер 1 (пины 9 и 10) 8 bit | Таймер 2 (пины 3 и 11) 8 bit | Таймер 2 (пины 3 и 11) 8 bit |
|:---:|:---:|:---:|:---:|:---:|
| | *Phase-correct PWM* | *Fast PWM* | *Phase-correct PWM* | *Fast PWM* |
| **1** | 31.4 кГц | 62.5 кГц | 31.4 кГц | 62.5 кГц |
| **2** | 4 кГц | 7.8 кГц | 4 кГц | 8 кГц |
| **3** | 490 Гц | 976 Гц | 980 Гц | 2 кГц |

*   Работа в автономном режиме благодаря модулю часов реального времени DS3231.
*   Энергонезависимая память: все настройки сохраняются после отключения питания.

Для сборки понадобятся (см. также схему):

| Компонент | Количество | Примечание |
| :--- | :---: | :--- |
| Arduino Nano | 1 | Протестирована только Nano |

## Логика работы 

Управление осуществляется одним **энкодером**:
*   **Вращение:** Навигация по меню / Изменение значений.
*   **Короткое нажатие (Click):** Выбор пункта / Вход в режим редактирования.
*   **Длинное нажатие (Hold):** Назад / Выход из режима редактирования / Сохранение.

### Главное меню
Главное меню состоит из 8 пунктов.

#### 1. ENABLE COLOR
Включение или выключение пользовательских цветов (всего доступно 6 по умолчанию). Активные цвета участвуют в расписании.
В начале скетча можно менять `#define MAX_COLORS 6`, до 20 цветов должно работать без проблем, не тестировал.

#### 2. SETUP COLOR
Настройка значений RGB (0-255) для выбранного активного цвета.
*   Вращением выберите цвет, кликом войдите в настройку.
*   Выберите курсором (`>`) компонент R, G или B и кликните.
*   Вращением измените значение яркости. Нажмите еще раз для фиксации.
*   Во время настройки лента светится текущим цветом.

#### 3. TIME COLOR
Установка временного интервала работы для каждого цвета.
*   `S` (Start) — время включения.
*   `E` (End) — время выключения.
*   Интервалы не должны пересекаться для одного дня. Если задано пересечение, появится предупреждение `Time overlap!`.
* Т.е. вся работа расписана на 24 часа, если нужно чтобы цвет горел через полночь, нужно настроить два Color до 00:00 и соответственно после.
* Например, нужно чтобы цвет горел с 23:00 до 01:00 следующего дня, нужно настроить Color1 start 23:00 end 00:00 и Color2 start 00:00 end 01:00. Естественно следующий день должен быть включен в расписании.

#### 4. TIMETABLE
Настройка дней недели, по которым будет работать автоматический режим.
*   Выберите день (Mon-Sun) и нажмите для переключения `yes` / `no`.
*   Короткое нажатие (Click) переключает состояние дня:
    *   **`yes`** — в этот день расписание работает. Лента будет включаться по заданным интервалам.
    *   **`no`** — в этот день расписание отключено. Лента светиться не будет, на экране в режиме RUN будет надпись **`Today Off Leds`**.


#### 5. CLOCK
Ручная установка времени и даты в модуле DS3231 (Устанавливается атоматически при заливке скетча). Можно использовать при замене батарейки или если по какой то причине время установилось не верно.

#### 6. TEST
Быстрый тест каналов. Поочередно включает Красный, Зеленый и Синий на полную яркость для проверки подключения транзисторов.

#### 7. RUN
**Основной режим работы.**
*   При входе запускается планировщик. На главном экране отображается текущее время, дата и статус ленты (Какой цвет активен и когда закончится / Какой цвет будет следующим).
*   **Короткое нажатие (Click):** Показать текущие значения RGB на 3 секунды.
*   **Длинное нажатие (Hold):** Остановить планировщик и выключить ленту (появится `JOB STOP`).

#### 8. DISPLAY
Управление подсветкой экрана для экономии энергии:
*   `OFF` (не используется, т.к. экран гаснет автоматически по таймеру)
*   `15MIN` — гаснет через 15 минут бездействия.
*   `1HOUR` — гаснет через 1 час.
*   `ALWAYS` — горит постоянно.

## Компиляция и загрузка прошивки

Важно! Отключайте внешнее питание при загрузке скетча (блок питания, если подключен).

### Зависимости (Библиотеки)
Перед компиляцией убедитесь, что в Arduino IDE установлены следующие библиотеки (Менеджер библиотек: *Скетч -> Подключить библиотеку -> Управлять библиотеками*):

1.  **LiquidCrystal_I2C** (by Frank de Brabander)
2.  **EncButton** (by AlexGyver)
3.  **GyverDS3231** (by AlexGyver)
4.  **GyverPWM** (by AlexGyver)

### Настройки платы
1.  Выберите плату: `Arduino Nano`.
2.  Процессор: `ATmega328P (Old Bootloader)` (если с новым не загружается).
3.  Порт: Выберите COM-порт вашей платы.
4.  Скопируйте код в окно Arduino IDE или откройте файл с расширением .ino.
Примечание: файл ino должен лежать в одноименной папке, например RGB.ino должен лежать в папке RGB.
Примечание: чтобы инвертировать поворот энкодера, нужно просто сменить знак: `int8_t change = -enc.dir();` // Можно убрать `-`, чтобы инвертировать, по умолчанию `int8_t change = enc.dir();`
5.  Нажмите кнопку "Загрузить" (стрелка вправо). Дождитесь окончания загрузки.
6.  Отключить от USB (!) если хотите сразу подключить к блоку питания, но можно не подключать блок питания, а потестировать работоспособность энкодера и управление через меню.

### Первый запуск
1.  После включения на экране появится заставка с версией прошивки и `RTC OK` или `RTC ERROR`.
2.  Если `RTC ERROR`, проверьте подключение модуля DS3231.
3.  Зайдите в меню **CLOCK** и установите правильное время (если оно по какой то причине отличается).

## Структура хранения в EEPROM

Все настройки (цвета, дни недели, таймаут экрана) хранятся в энергонезависимой памяти. При изменении любого параметра в меню автоматически происходит сохранение через 1-2 секунды.
